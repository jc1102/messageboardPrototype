## To Get the Prototype start

For Windows user:

Install MogoDB:


Download from https://www.mongodb.com/community 


start the database:


```bash
cd C:\data\db
mkdir data
cd data
mkdir db

cd C:\Program Files\MongoDB\Server\3.2\bin
mongod

```


Enter Mongo Shell:


```bash
mongo
```


start the back-end


```bash
cd back-end
npm install
node server.js
```


start the front-end


```bash
cd front-end
npm install
bower install
gulp server
```
